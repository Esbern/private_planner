# Private Planner

A local-first system for managing personal media, travel, and aesthetics using an Obsidian vault.

Setup
- Ensure Python 3.11+.
- The CultureDB Obsidian vault lives under `CultureDB/` by default.

Initialize
```
python init.py
```

Notes
- `.env` is created at project root with placeholders.
- Set `CULTUREDB_ROOT` in `.env` to override vault path.
